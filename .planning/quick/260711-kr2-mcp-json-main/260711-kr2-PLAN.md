# Quick Task 260711-kr2: `.mcp.json` を追跡対象から外す

## 背景

プロジェクトトップレベルの `.mcp.json` は GSD インストールが書き込む MCP サーバ設定で、
`/home/yuta/...` や nix ストアパスといったマシン固有の絶対パスを含む。共有すべきでない
にもかかわらず git に追跡され、`origin/main`（コミット `1dc3a77`）のツリーにも存在する。

既存の `.gitignore` は同じ理由（マシン固有の絶対パスを焼き込むため）で `.claude/` を
ローカル限定として無視している。`.mcp.json` も同カテゴリ。

## タスク

1. `git rm --cached .mcp.json` で追跡を解除（ワーキングツリーのファイルは残す）。
2. `.gitignore` の GSD セクションに `.mcp.json` を追加し、再追跡を防止。
3. 変更をアトミックにコミット。

## verify

- `git ls-files .mcp.json` が空を返す（追跡されていない）。
- `git check-ignore .mcp.json` が `.mcp.json` を返す（無視される）。
- `.mcp.json` がワーキングツリーに物理的には残っている。

## done

`.mcp.json` が git 管理外になり、`.gitignore` で無視され、この release ブランチが
main へマージされた時点で main のツリーから外れる。

## 注意 / スコープ外

過去履歴（`1dc3a77`）に残るブロブの完全消去（filter-repo + force push）は共有履歴の
書き換えとなりリスクが高いため本タスクのスコープ外。必要なら別途明示的に依頼すること。
