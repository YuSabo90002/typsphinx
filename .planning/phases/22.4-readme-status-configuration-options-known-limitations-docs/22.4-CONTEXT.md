### Configuration Options 節（README:203-211）

- **D-05:** 全 12 件の列挙はしない。「主要なもの」と読み手に分かる見出し・文言に改め、全一覧は docs へ誘導する部分列とする。理由: 乖離の本質は「5 件しかないように見える」ことであり、件数を増やすと今後 `typst_*` を増減するたび README も更新が要る（Phase 22.2 で実際に 2 件削除している）。
