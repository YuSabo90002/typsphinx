<objective>
Ship the GATE-01 real-compile acceptance fixture that proves BLK-02 + BLK-03 + the SC#3 admonition-

→ typst.compile() → pypdf`, never string-agreement asserts alone). This is the single combined fixture

appears exactly once (not duplicated into the auto-outline), (2) a `.. contents::` renders box-less,
(3) address + poem line_blocks render with every line break preserved, and (4) the existing admonition
titles — including a multi-child `.. admonition:: Custom *Title*` (the Pitfall-1 regression case) —
still render correctly after the `visit_title` generalization.

`TestTopicLineBlockRenderGate` class in `tests/test_pdf_render_gate.py`. Depends on BOTH 13-01 (topic /
visit_title) and 13-02 (line/line_block) being landed — the fixture exercises both.

Constraint (Pitfall 4, mandatory): the fixture MUST NOT use `.. epigraph::` (or any attributed
block_quote) — those hit two unrelated pre-existing bugs (a hard fatal with attribution; a literal-
source leak without) that would fail this gate for reasons outside BLK-02/BLK-03. Render the "epigraph"
shape as a plain top-level `line_block` under a titled section instead (verified compile-safe).
</objective>
