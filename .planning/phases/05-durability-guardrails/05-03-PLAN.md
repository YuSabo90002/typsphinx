<objective>
Close DUR-02 by creating a new standalone `.github/workflows/drift.yml`: a weekly,
non-blocking, forward-drift detector that intentionally re-resolves to the latest allowed
dependency versions, exercises them against the test suite and the historically-fragile
PDF path, and on failure files/updates a single deduplicated GitHub Issue so drift is a
visible, persistent signal rather than a buried red X.

Purpose: this milestone's root cause was a silent multi-year dependency drift (a fresh
resolution jumped to typst 0.15 and broke PDF compilation via the `kai` symbol). DUR-02
installs an early-warning alarm for exactly that class of forward drift — it runs latest
deps on a schedule and shouts (via an Issue) the moment they break, long before a human
would notice, and does so WITHOUT ever blocking a merge (D-04/D-07) or altering the
shipped lock (D-05).

Output: 1 new file (.github/workflows/drift.yml).
</objective>
