## Flagged assumption (spec-less edge probe — TODO-01 unclassified row, surfaced, not dropped)

The fallback edge probe could not classify TODO-01's edge category. The one genuine open behavior
question (16-RESEARCH.md Open Question 1 / Assumption A2) is whether `visit_todo_node` gates on
`config.todo_include_todos`. **Planner decision: YES, gate on it** — every official Sphinx builder
(HTML, LaTeX, text, man, texinfo) raises `nodes.SkipNode` when it is False, and diverging would be a
fidelity bug of exactly the class this milestone hunts. This is a deliberate, documented choice
within Claude's discretion (CONTEXT.md does not lock it); the disabled-case test below pins it.
