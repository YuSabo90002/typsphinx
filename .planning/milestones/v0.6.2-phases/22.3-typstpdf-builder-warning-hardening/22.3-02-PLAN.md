module keeps two test functions: the include/image half becomes a full GREEN ablation, and the
template half is positioned as its RED-only differential.

  unused after the literal asserts are removed, each test attaches it to the failure message of an

progressively neutralized copies) and merging would make the differential relationship implicit
rather than stated.

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| test suite → typst-py / Rust compiler | Real `typst.compile()` results drive CI pass/fail. The compiler's diagnostic TEXT is an uncontracted upstream surface; this plan removes every dependency on it. |
