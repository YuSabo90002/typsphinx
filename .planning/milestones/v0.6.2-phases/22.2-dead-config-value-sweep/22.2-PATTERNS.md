## Shared Patterns

### GATE-01 real-compile fixture (applies to `tests/fixtures/<new>/` + `tests/test_<new>_gate.py` + the D-12 example test)

**Source:** `tests/test_preview_smoke_gate.py` (full file) + `tests/fixtures/preview_smoke/conf.py`
**Apply to:** all three new/extended integration-test surfaces in this phase (D-09's fixture+gate,
D-12's example-build test)

```python
try:
    import typst
    TYPST_AVAILABLE = True
except ImportError:
    TYPST_AVAILABLE = False

@pytest.mark.skipif(not TYPST_AVAILABLE, reason="typst-py is required for ...")
def test_...(fixture_dir, temp_build_dir):
    result = subprocess.run(
        [sys.executable, "-m", "sphinx", "-b", "typst", str(fixture_dir), str(temp_build_dir)],
        capture_output=True, text=True,
    )
    assert result.returncode == 0, f"sphinx-build failed:\nstdout: {result.stdout}\nstderr: {result.stderr}"
    index_typ = temp_build_dir / "index.typ"
    pdf_output = temp_build_dir / "index.pdf"
    typst.compile(str(index_typ), output=str(pdf_output))  # uncaught = loudest failure signal
    assert pdf_output.exists() and pdf_output.stat().st_size > 0
```
