## Worktree provisioning (MANDATORY — read before running anything)

This executor runs inside an isolated git worktree (`test -f .git` is true). Before running any
command:

1. `env -u VIRTUAL_ENV -u UV_PROJECT_ENVIRONMENT uv sync --extra dev`
2. Run **every** subsequent command through `uv run` (e.g. `uv run pytest …`).

Without this, pytest imports the UNCHANGED main-tree package, the gate compiles the OLD emitted
output, and it stays red after a correct fix (or, worse, green against the wrong code). See
`CLAUDE.md` § "Worktree-isolated execution".
