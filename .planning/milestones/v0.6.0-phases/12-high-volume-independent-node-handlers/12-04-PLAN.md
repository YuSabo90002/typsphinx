<objective>
Land the four trivial structural node handlers (BLK-01/04/05/06), all small additive `translator.py`
handlers reusing already-proven idioms, plus a real-compile GATE-01 fixture. BLK-01 (`transition`)
emits a `line(length: 100%)` horizontal rule then `raise nodes.SkipNode` (a genuine content gap — it
renders nothing today); BLK-04 (`glossary`) is a transparent pass-through to the underlying definition
list; BLK-05 (`tabular_col_spec`) is a bare `raise nodes.SkipNode` that safely drops the LaTeX-only
`.. tabularcolumns::` hint; BLK-06 (`abbreviation`) appends the expansion inline as "term (expansion)"
via dummy-Text-node delegation, stateless on every occurrence (D-08).

Purpose: these complete the highest-frequency remaining dropped structural nodes. BLK-06 is a genuine
content gap (the `explanation` attribute is completely dropped today); the security-relevant detail is
that the expansion text is author-controlled RST and MUST be routed through the existing `visit_Text`
escaping regime (dummy-node delegation), never a raw f-string interpolation and never `node.astext()`
(V5 Input Validation, Pitfall 7). BLK-04's glossary fixture compiles because the `depart_term`
anchor fix already landed in Plan 12-02 (wave 2 < wave 4).

Output: `visit_transition`/`depart_transition`, `visit_glossary`/`depart_glossary`,
`visit_tabular_col_spec`, `visit_abbreviation`/`depart_abbreviation`, one fixture project, and one
real-compile test class.

**Requirements coverage:** BLK-01 (D-07), BLK-04 (D-07), BLK-05 (D-07), BLK-06 (D-07 + D-08). GATE-01
standing bar extended (SC#5).

**Wave note:** `depends_on: [12-03]` is a file-ownership sequencing dependency (shared `translator.py`
+ `test_pdf_render_gate.py`), not a logical one — the BLK group is independent of the other three. BLK-04

label), which the wave ordering already satisfies.

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| author-controlled RST attribute (`abbreviation.explanation`) → generated Typst source | the abbreviation expansion text is author-controlled and crosses into generated Typst; markup-special characters could corrupt or crash the compile if unescaped |
| generated Typst → typst.compile() → PDF → pypdf | the render gate samples this boundary to prove the abbr expansion renders and no tabularcolumns spec leaks |
